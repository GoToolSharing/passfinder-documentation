# Numbers

The `--numbers` flag adds numbers from 0 to 20 (default) 

```bash
> passfinder company -n passfinder --numbers

```

The `--numbers-range` flag is used to adjust the range size.

```bash
> passfinder company -n passfinder --numbers --numbers-range 5

passfinder
passfinder0
passfinder1
passfinder2
passfinder3
passfinder4
passfinder5
```
