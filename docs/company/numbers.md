# Numbers

The `--numbers` flag adds numbers to the company name. It waits for a maximum value and then iterates on it.

```bash
> passfinder company -n passfinder --numbers 5

passfinder
passfinder0
passfinder1
passfinder2
passfinder3
passfinder4
passfinder5
```

```bash
> passfinder company -n passfinder --numbers 10
```
