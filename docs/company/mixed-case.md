# Mixed-case

The `--mixed-case` flag allows you to mix the case of a password, alternating between upper and lower case.

```bash
> passfinder company -n passfinder --mixed-case

[...]
PAssFINDER
PAsSfinder
PAsSfindeR
PAsSfindEr
PAsSfindER
PAsSfinDer
PAsSfinDeR
PAsSfinDEr
PAsSfinDER
PAsSfiNder
[...]
```
